🍗 KFC Sales & Performance Analysis (Power BI Project)
📌 Project Overview

This project focuses on analyzing KFC (Kentucky Fried Chicken) sales and performance data to generate meaningful business insights. The goal is to transform raw data into interactive and visually appealing dashboards using Power BI.

The project includes data cleaning using Python (EDA) and data modeling with DAX measures to provide accurate and actionable insights for decision-making.

🎯 Project Objectives
Analyze total sales and revenue trends
Evaluate profitability across products and categories
Identify top-performing and underperforming products
Understand customer purchasing behavior
Compare regional/outlet-wise performance
Build interactive dashboards for better visualization
Support data-driven decision-making
🛠️ Tools & Technologies Used
Power BI – Dashboard creation & visualization
Python (EDA) – Data cleaning and preprocessing
DAX (Data Analysis Expressions) – Calculations & measures
Excel / Dataset Source – Raw data handling
📊 Key Features of the Dashboard
📈 Total Sales Report
💰 Total Profit Report
📊 Category-wise and product-wise performance
🌍 Regional/Outlet comparison
🔍 Interactive filters and slicers
📉 Trend analysis over time
🧹 Data Cleaning & Preparation
Performed Exploratory Data Analysis (EDA) using Python
Handled missing values and duplicates
Standardized and formatted data
Prepared dataset for Power BI modeling
🧠 DAX Measures Created

Some important DAX calculations used:

Total Sales
Total Profit

Year-over-Year Sales
Category-wise Performance Metrics
